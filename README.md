# Vblog

## 需求

用户管理：
   - 用户注册和登录功能，允许用户创建和管理个人账户。
   - 用户个人资料管理，包括头像、昵称、个人简介等信息的编辑和更新。   
   - 用户角色和权限管理，例如管理员、作者、普通用户等，允许管理员对用户进行权限控制。

文章管理：
   - 文章发布和编辑功能，允许作者创建和编辑文章，包括标题、内容、分类和标签等信息。
   - 文章删除和归档功能，允许作者或管理员删除或归档不需要展示的文章。
   - 文章草稿功能，允许作者保存未完成的文章并随时编辑。
   - 文章阅读量统计，记录每篇文章的阅读次数。

评论管理：
   - 评论发布和回复功能，允许读者对文章发表评论，并允许作者或其他读者回复评论。
   - 评论审核功能，允许管理员对评论进行审核和管理，以过滤不合适的内容。

搜索功能：
   - 提供关键词搜索功能，允许读者通过关键词搜索相关的文章
   - 展示搜索结果页面，包括匹配的文章列表和相关信息。

## 原型

首页：
   - 展示最新发布的文章列表，包括文章标题、作者、发布日期和阅读量。
   - 提供搜索框，允许读者通过关键词搜索文章。
   - 提供分类和标签导航，方便读者浏览相关文章。

文章详情页：
   - 展示文章的详细内容，包括标题、作者、发布日期、阅读量和文章内容。
   - 允许读者发表评论和回复评论。
   - 显示相关文章推荐，根据分类或标签推荐相关的文章。

用户注册和登录页：
   - 提供用户注册功能，包括用户名、密码和确认密码。
   - 提供用户登录功能，允许已注册用户登录博客系统。

用户个人资料页：
   - 展示用户的个人资料，包括头像、昵称、个人简介等。
   - 允许用户编辑和更新个人资料。

文章发布和编辑页：
   - 允许作者发布新的文章，包括标题、内容、分类和标签等。
   - 允许作者编辑已发布的文章，包括标题、内容、分类和标签等。

管理员后台：
   - 提供管理员登录功能，允许管理员管理博客系统。
   - 允许管理员管理用户，包括用户角色和权限管理。
   - 允许管理员管理文章，包括发布、编辑、删除和分类管理等。
   - 允许管理员管理评论，包括回复、删除和审核等。

搜索结果页：
   - 展示根据关键词搜索到的文章列表，包括文章标题、作者、发布日期和阅读量。

分类和标签页：
   - 展示不同分类和标签下的文章列表，方便读者浏览相关文章。

404页面：
   - 当访问的页面不存在时，显示404页面，提供返回首页或其他页面的链接。


## 前后端分离的博客项目，用到的技术：
+ API Server
  + DDD：使用领域驱动设计
  + Ioc：使用ioc/container，做解耦、注入、依赖倒置
  + Gin：使用Gin框架创建HTTP路由，处理博客系统的请求和响应
  + MySQL：使用GORM库定义博客系统的数据库模型
+ Vue3 + ArcoDesign

## 分层介绍
各目录的作用：
- /apps/：业务处理模块
  - /blog/：文章管理模块
  - /comment/：评论管理模块
  - /token/：鉴权管理模块
  - /user/：用户管理模块
    - /api/
      - /http.go/：用户对外暴露的接口
    - /impl/
      - /impl.go/：用户业务实现
    - /enum.go/：枚举
    - /interface.go/：用户接口定义
    - /model.go/：用户数据结构定义
- /common/：通用公共包
  - /meta.go/
- /conf/：配置管理
  - config.go：全局配置文件
  - load.go：加载配置
- /etc/：程序加载的配置文件目录
- /exception/：异常处理
  - /bussiness/：业务异常
  - /exception/：异常定义
- /ioc/：对象托管
- /middleware/：中间件
  - /auth.go/：鉴权
- /protocol/：协议服务
- /response/：web响应
- /test/：测试模块
- /web/：项目前端
